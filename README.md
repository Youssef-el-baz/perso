## Commandes Git pour pousser vers GitHub

Pour pousser votre portfolio vers `git@github.com:Youssef-el-baz/portfoliopub.git` :

```bash
# Dans le dossier portfolio
cd portfolio

# Initialiser git si nécessaire
git init

# Ajouter le remote
git remote add origin git@github.com:Youssef-el-baz/portfoliopub.git
# OU si le remote existe déjà :
git remote set-url origin git@github.com:Youssef-el-baz/portfoliopub.git

# Ajouter tous les fichiers
git add .

# Créer un commit
git commit -m "Update portfolio with all skills organized by categories"

# Pousser vers GitHub
git branch -M main
git push -u origin main
```

Ou utilisez les scripts fournis :
- Linux/WSL : `bash push_to_github.sh`
- Windows : `push_to_github.bat`
